HTML

O que é HTML?

HTML é uma linguagem de marcação de hipertexto, resumindo e sem muita enrolação é a linguagem que você vai utilizar para fazer a estrutura da sua página, HTML que hoje (26/05/2020) está na sua versão 5, é baseado em tags.


O que são tags?

Tags são basicamente rótulos que indicam ao navegador a forma de apresentar o conteúdo, em HTML, praticamente tudo é uma tag, por exemplo, para você colocar um parágrafo você usa a tag "p".


Afinal como escrever essas tags?

As tags do HTML sempre tem que ser abertas e depois fechadas, como esse exemplo:


<p>conteúdo</p>

Porém existem tags que se "autofecham" exemplo a tag "br" que serve para dar uma quebra de linha:


<br>

Aqui você pode ver uma lista com todas as tags do HTML5, mas fique tranquilo que de cara você não precisa decorar todas, outra coisa bem importante são os atributos, são resumidamente as configurações da tag, lá você vai definir oque chamamos de classe e id (classe é como se fosse um grupo de elementos com o mesmo nome que os caracterizam, e id é um nome "único" para cada elemento), as principais tags e atributos são esses: clique aqui.

O que são divs?

Pode parecer algo um pouco abstrato no começo, divs são basicamente "containers/caixas" com um espaço dentro, nela você organiza o conteúdo, na maioria das vezes colocar um conteúdo dentro ou fora de uma div não vai mudar muita coisa, mais é sempre bom optar pelas "boas práticas", aliás é um padrão seguido em quase todos os sites.

Ignore os nomes, oque importa é que você pode ver que cada quadrado é um espaço na tela, ou seja, a div não é um elemento visível, apenas um espaço reservado para tal parte da sua página.


Entenda de uma vez oque são os atributos "class" e "id"

Vamos começar com exemplos do mundo real, provavelmente você sabe oque é uma cadeira certo? Ou seja, é como se você desse um nome pra esse grupo de objetos, isso é oque é classe em HTML, você está classificando um grupo de coisas com o mesmo nome. Agora vamos explicar sobre o id, imagine que você tem uma cadeira na sua casa e precisa informar ao seu amigo qual é a sua cadeira para que ele compre uma igual, você provavelmente vai falar a marca, ou o modelo, ou até mesmo a cor, isso é id! É simplesmente uma especificação de cada elemento. Todo elemento pode ter uma class e um id, exemplo:


<p class="titulos" id="titulo-1">conteúdo</p>

Por que é importante atribuir classes e id´s à um elemento HTML?

Há alguns motivos:

  1. Organização;
  2. Interação com CSS fica mais fácil;
  3. Interação com Javascript também fica mais fácil;

Padrão Estrutural

Após aprender tudo isso, você está apto à aprender a estrutura padrão de quase todos os sites da web, você não necessariamente tem que seguir exatamente como o padrão, mas, há algumas tags e classes que são utilizadas normalmente para uma finalidade.




Muitas vezes você vai se deparar com páginas que dividem o conteúdo com muitas divs, isso é bem comum, para identificar cada div utilizamos o id e a classe para compreender o código.


Não se sente tão seguro?

Isso é normal, no começo HTML pode ser um pouco estranho e não tão óbvio, mais aos poucos você se acostuma facilmente. Caso tenha alguma dúvida (ou esqueceu alguma tag), na maioria das vezes uma rápida pesquisa no google resolve.
